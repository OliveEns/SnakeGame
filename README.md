# ESP32-C3 贪吃蛇游戏

基于北京邮电大学电子工程学院《嵌入式智能编程》课程 ESP32-C3 开发板的贪吃蛇游戏，使用 ST7789 240x240 SPI 显示屏显示。


## 硬件配置

| 组件 | 连接引脚 | 说明 |
|------|----------|------|
| ST7789 显示屏 MOSI | GPIO5 | SPI 数据引脚 |
| ST7789 显示屏 CLK | GPIO18 | SPI 时钟引脚 |
| ST7789 显示屏 DC | GPIO6 | 数据/命令选择 |
| ST7789 显示屏 CS | GPIO4 | 片选引脚 |
| ADC 按键 | GPIO3 | 方向控制 |
| 数字按键 | GPIO9 | 返回/确认键 |
| LED 指示灯 | GPIO19 | 游戏状态指示 |

### ADC 按键阈值

ADC 按键使用分压电路实现多个按键功能，阈值配置如下：

| 阈值 | 功能 |
|------|------|
| 0-800 | 向下 |
| 800-1600 | 向左 |
| 1600-2400 | 向右 |
| 2400-3100 | 向上 |
| 3100-3900 | 确认 |
